## 🛠️ 技术栈

基于最新的技术栈构建，确保你的应用具备**高性能、可扩展、易维护**的特性：

- **前端**: React 19 + React Router v7 + Vite 7
- **后端**: Hono + Drizzle ORM + PostgreSQL
- **部署**: Cloudflare Workers + Pages
- **样式**: Tailwind CSS v4 + Shadcn UI
- **类型安全**: 全栈 TypeScript

## 📦 30+ 核心模块

采用 Monorepo 架构，**30+ 个独立模块**，每个模块职责清晰，易于维护和扩展：

- UI 组件库、API 服务层、数据库操作
- 认证系统、支付处理、AI 集成
- 国际化、SEO 优化、分析统计
- 邮件服务、存储服务、错误处理

## ⚡ 快速开始

只需 3 步，立即开始：

<Steps>
  <Step>
    **克隆项目**

    ```bash
    git clone <repository-url>
    cd seeksaas-www-v7-react-router
    ```
  </Step>

  <Step>
    **安装依赖**

    ```bash
    pnpm install
    cp .env.example .env.local
    # 配置你的环境变量
    ```
  </Step>

  <Step>
    **启动开发**

    ```bash
    pnpm dev
    ```

    访问 `http://localhost:5173` 查看你的应用！
  </Step>
</Steps>
